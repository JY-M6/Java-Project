**惠民商城 (2026.2 - 2026.5) | 后端开发**
* **技术栈**：Spring Cloud Alibaba、Nacos、Gateway、MyBatis-Plus、MySQL、Redis Cluster、Redisson、RabbitMQ、Elasticsearch、Sentinel、Seata、Docker
* **项目简介**：基于 Spring Cloud 微服务架构构建的高可用B2C电商平台。系统根据业务边界拆分为商品、订单、购物车、支付、用户等核心微服务，模拟应对高并发与海量数据检索场景。
* **核心职责**：
  1. **微服务架构与容错**：基于 Nacos 实现服务注册与统一配置中心；集成 Sentinel 配置自定义流控规则与熔断降级策略，保障核心交易链路在高并发下的可用性。
  2. **海量数据搜索优化**：引入 Elasticsearch 替代传统数据库模糊查询，通过自定义分词器实现海量商品的全文检索、多条件聚合过滤及高亮显示，大幅提升检索性能。
  3. **高可用缓存与锁机制**：主导搭建 Redis 分片集群，设计缓存主动更新与兜底淘汰策略；引入 Redisson 分布式锁解决并发场景下的缓存击穿及高并发库存扣减防超卖问题。
  4. **异步削峰与分布式事务**：使用 RabbitMQ 改造订单生成链路，实现订单超时自动取消、库存异步释放等业务解耦；集成 Seata (AT模式) 保证跨服务（订单生成与库存扣减）的数据最终一致性。

**智能售货机运营管理平台 (2025.7 - 2025.10) | 后端开发**
* **技术栈**：Spring Boot、MyBatis-Plus、MySQL、Redis、EMQX (MQTT)、Vue3、Uniapp、Docker
* **项目简介**：面向多终端（Web管理端、App运营端、设备机端）的物联网售货机管理系统。涵盖设备点位管理、投放/撤机工单调度、商品库存监控与销售数据分析。
* **核心职责**：
  1. **核心业务开发**：设计并实现后端 RESTful API，完成售货机设备模型搭建及“区域-点位-设备-货道”的复杂关联关系映射。
  2. **工单调度流转**：设计并开发设备投放、补货、维修等工单的完整生命周期流转逻辑（待办、接受、执行、完成），配合 App 端提供高效的现场运维支持。
  3. **物联网设备通信**：基于 EMQX 消息服务器，利用 MQTT 协议实现云端与售货机主板的远程指令下发与状态数据上报。
  4. **缓存与性能优化**：引入 Redis 实现热点数据（如商品详情、字典数据）的缓存，有效降低数据库查询压力，将核心接口响应时间降低60%以上。

**TT智能学习辅助系统 (2025.3 - 2025.6) | 全栈开发**
* **技术栈**：Spring Boot、MyBatis、Vue3、ElementPlus、JWT、AOP、ThreadLocal、Aliyun OSS
* **项目简介**：符合现代企业级规范的前后端分离后台管理系统，提供标准的部门管理、员工管理、权限校验与数据统计功能。
* **核心职责**：
  1. **工程化架构搭建**：采用 Maven 多模块工程进行分层设计，剥离公共工具类与实体POJO，降低代码耦合度；统一封装全局响应结果对象(Result)。
  2. **统一登录认证与上下文**：基于 JWT 实现无状态的登录鉴权，结合 Spring 拦截器(Interceptor)完成请求拦截；巧妙使用 ThreadLocal 存储当前线程的用户 ID，实现 Service 层安全的上下文变量传递。
  3. **AOP操作日志与异常处理**：通过自定义注解结合 Spring AOP 面向切面编程，实现核心业务接口的无侵入式操作日志记录；构建全局异常处理器(GlobalExceptionHandler)，避免代码冗余。
  4. **第三方服务集成**：对接 Aliyun OSS 对象存储服务，实现员工头像及资料等文件的云端上传与持久化存储。
  5. **前端界面构建**：运用 Vue3 组合式 API 与 ElementPlus 组件库，实现动态表格展示、多条件分页查询及复杂表单的交互与数据校验。
